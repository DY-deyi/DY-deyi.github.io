Markdown基础语法与其编辑器使用

发表于 2021-05-09 更新于 2021-05-11 分类于 标记语言 阅读次数：

Markdown?

Markdown和HTML是一种 [标记语言] ，通常为程序员们所使用。当我们想用HTML写一个网页时，通常要使用大量的标签来完成，而这对于仅仅想要记录或者笔记功能的我们，非常不方便；而Markdown作为一款轻量级标记语言，很好的解决了这个问题。

Markdown拥有更简洁的语法格式，对于记录和笔记来说，有着别的语言不可比拟的优势。

Markdown应用

  • 编撰电子书
  • 网站帮助文档或是论坛上发表消息，如github，简书，hexo等
  • 记笔记
  • 写博客

Markdown优势

在写 markdown 优势前，先聊一下其他记录或编写文档的方式，我们最熟知的就是 windows 上的 word 和 macOS 上的 Pages；

过去乃至如今，大部分的人还是使用这两种写文稿，当然能被大多数人所认可，绝对优点多多，但今天我们是来聊 markdown 的优势的，那就只能先委屈这两位了，我们来细数一下这两种格式的文稿在我们日常生活中所出现的错误或者劣势：

  • 对于系统有着要求，对于 windows 的人打不开.pages文件 而对于 macOS 的人打不开.doc ,.docx 文件；
  • 使用旧版和新版的word打开同一个文件，可能格式出现很大的区别；
  • 使用盗版的office打开文件，很容易出现错误；
  • 在不同的电脑上，即使使用同一版本的word也有可能出现格式不同的情况。

这时候就该我们的 markdown 出场的时候了，markdown 优点如下：

  • 语法简单易学。
  • 纯文本，所以兼容性极强，可以用所有文本编辑器打开。
  • 让你专注于文字而不是排版。
  • Markdown 编写的文档可以导出 HTML 、Word、img、PDF、Epub 等多种格式的文档。
  • Markdown 的标记语法有极好的可读性。
